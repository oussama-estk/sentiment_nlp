# NLP Sentiment Analysis Project

This project implements a Sentiment Analysis model using Natural Language Processing (NLP) techniques. It classifies text reviews as either **Positive** or **Negative**. 

The project is built with Python, Scikit-Learn, and Flask, and is ready for deployment on Render.
